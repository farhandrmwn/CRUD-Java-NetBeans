# CRUD-Java-NetBeans
CRUD Java dengan NetBeans dan MySql

  Cara Menjalankan:

1.Buka XAMPP Control Panel.

2.Jalankan MySql dan Apache.

3.Buka localhost/phpmyadmin di browser anda.

4.Import db_siswa.

5.Buka Project yang telah anda download dengan NetBeans.

6.Jalankan Project.

7.Selesai.
